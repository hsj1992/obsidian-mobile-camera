# Documentation

This folder contains additional documentation for the Mobile Camera & QR Scanner plugin.

## Files

- **[advanced.md](advanced.md)** - Advanced usage guide, troubleshooting, and tips
- **images/** - Screenshots and demo GIFs (to be added)

## Adding Screenshots

When adding screenshots or GIFs, place them in the `images/` folder and reference them in the documentation:

```markdown
![Description](images/screenshot.png)
```
